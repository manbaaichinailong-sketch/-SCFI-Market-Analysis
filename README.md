# SCFI Container Shipping Market Intelligence (2020-2026)

A portfolio-grade analysis of the Shanghai Containerized Freight Index (SCFI), designed for shipping, liner, freight-forwarding and maritime internship applications.

## What makes this project different

This is not just an index chart. The project combines:

- 79 monthly reference observations from January 2020 to July 2026
- Current public SCFI snapshot as of 17 July 2026
- MoM and YoY change, 3M/12M moving averages, volatility and drawdown
- Annual cycle summary and market-regime classification
- Event study covering COVID-19, congestion, Red Sea rerouting, tariffs, port fees and the 2026 fuel/geopolitical shock
- Official route structure and calculation framework
- A demand/effective-capacity driver model
- Data-quality and licensing disclosure
- Chinese and English interview pitches
- Reproducible Python charts and CSV outputs

## Key findings

1. **2021-2022 was an effective-capacity crisis, not simply a cargo-volume boom.** Goods demand, port queues, container shortages and low schedule reliability reinforced one another.
2. **The late-2022 collapse shows how quickly spot rates can normalize.** Demand and inventory conditions weakened while congestion released capacity back into the network.
3. **2023 was a structural low.** Weak cargo demand met rapid fleet growth and newbuilding deliveries.
4. **The 2024 rally was disruption-led.** Red Sea/Cape rerouting increased voyage distance and absorbed vessel-days, while front-loading and congestion tightened slots.
5. **Policy changes move shipment timing.** The 2025 tariff pause and port-fee changes created short-lived capacity squeezes without necessarily improving full-year trade demand.
6. **The 2026 rally combines cost and capacity channels.** Fuel, war-risk surcharges, route disruption and importer anxiety lifted rates, although the index remains below the 2022 record.

## Data scope and honesty

SCFI is published weekly by the Shanghai Shipping Exchange. The complete licensed weekly history is **not reproduced** in this repository.

The monthly dataset is a **reference curve reconstructed from public charts and published anchors** for learning, visualization and portfolio use. Each row contains a `Data Status`, `Source` and `Source URL`. Do not call the entire CSV an official raw dataset.

Latest public snapshot used in the dashboard:

- One-year change: +93.38%
- Date: 17 July 2026
- Source: Trading Economics

## Run the Python analysis

```bash
cd python
python -m pip install -r requirements.txt
python scfi_analysis.py
```

The script validates the input, calculates metrics, exports analytical CSV files and regenerates five charts.

## Excel workbook

The workbook contains nine analytical sheets:

1. **Dashboard** - current KPIs, watchlist and four charts
2. **Monthly Data** - source-audited data and formula-driven metrics
3. **Annual Summary** - market cycle by year
4. **Event Study** - before/after impact and transmission mechanism
5. **Route Structure** - official SCFI route/weight framework
6. **Driver Model** - demand, effective supply, fuel and carrier-management factors
7. **Data Quality** - licensing and source disclosure
8. **Interview Notes** - Chinese/English pitches and common questions
9. **Latest Snapshot** - current public indicators and source

## SCFI vs CCFI vs BDI

| Index | Main market | Rate type | Geographic scope | Main analytical use |
|---|---|---|---|---|
| SCFI | Container liner shipping | Shanghai export spot rates | Shanghai to major routes | Short-term container-rate momentum |
| CCFI | Container liner shipping | Broader spot + contract level | Major China export ports | More macro and less volatile benchmark |
| BDI | Dry bulk tramp shipping | Vessel freight/time-charter assessments | Global dry bulk routes | Iron ore, coal, grain and bulk-cycle analysis |
