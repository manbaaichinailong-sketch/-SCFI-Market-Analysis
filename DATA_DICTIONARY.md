# SCFI Data Dictionary

This document explains the fields used in the monthly reference dataset and
the analytical outputs generated by `python/scfi_analysis.py`.

## Input dataset

File: `data/SCFI_Monthly_Reference_2020_2026.csv`

| Field | Type | Description |
|---|---|---|
| Date | date | Monthly observation date in ISO format |
| SCFI | number | Shanghai Containerized Freight Index reference value |
| MoM Change | decimal | Month-over-month percentage change |
| YoY Change | decimal | Year-over-year percentage change |
| 3M Moving Avg | number | Three-month rolling average |
| 12M Moving Avg | number | Twelve-month rolling average |
| 12M Ann. Volatility | decimal | Annualized standard deviation of monthly returns |
| Year | integer | Calendar year |
| Month | integer | Calendar month |
| Market Phase | category | Low, Normal, Elevated, High or Extreme regime |
| Event Tag | text | Major market event associated with the observation |
| Data Status | category | Indicates whether the value is a public anchor or reconstruction |
| Source | text | Human-readable source description |
| Source URL | URL | Link used for source review |

## Generated analytical fields

File: `data/SCFI_Analysis_Output.csv`

| Field | Description |
|---|---|
| MoM | Month-over-month return calculated with `pct_change()` |
| YoY | Change from the observation twelve months earlier |
| MA_3M | Three-month moving average |
| MA_12M | Twelve-month moving average |
| Rolling_12M_Ann_Vol | Rolling monthly volatility annualized by the square root of 12 |
| Running_Peak | Highest SCFI value observed up to that date |
| Drawdown | Percentage decline from the running peak |
| Market_Phase | Index-level regime based on documented thresholds |

## Data-use note

The monthly series is a reference curve reconstructed from public charts and
published anchors. It is intended for learning and portfolio analysis and is
not a substitute for the licensed weekly SCFI history.
