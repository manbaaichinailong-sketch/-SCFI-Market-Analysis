# SCFI 项目使用说明（中文）

## 你在简历上怎么写

**上海出口集装箱运价指数（SCFI）市场分析项目｜Excel / Python**

- 整理并分析 2020—2026 年 79 条月度参考数据，计算环比、同比、3/12 月移动平均、波动率与回撤。
- 建立年度周期、关键事件和供需传导分析，解释疫情供应链危机、2022 年回落、红海绕航、关税抢运及 2026 年燃油/地缘冲击。
- 使用 Excel 制作市场仪表盘，并用 Python 完成数据校验、指标计算和图表自动生成。
- 区分 SCFI、CCFI 与 BDI 的市场口径、货种结构和运力管理逻辑。

## 面试最重要的三句话

1. SCFI 上涨不能只解释成“货变多了”，还要分析港口拥堵、绕航、班期可靠性、空班和集装箱周转对有效运力的影响。
2. 2022 年下半年快速下跌说明即期运价对需求正常化和拥堵缓解非常敏感。
3. SCFI 是上海出口集装箱即期市场，BDI 是干散货市场，两者的货种、商业模式和运力调节方式不同。

## 数据口径怎么回答

不要说：“我拿到了上海航运交易所 2020—2026 全部官方原始数据。”

正确说法：

> 我使用公开图表和已发布锚点重构了月度参考曲线，同时对每一行标注数据状态和来源。项目主要用于作品集分析；完整周度授权数据不在仓库中。

## 这个项目能证明什么

- 你会用 Excel 处理真实行业数据；
- 你能把指数变化翻译成航运供需逻辑；
- 你知道数据来源与口径不能乱说；
- 你能用 Python 让分析可复现；
- 你对班轮公司、船代、货代、航运市场实习有可展示的作品。
